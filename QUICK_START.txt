📱 تطبيق محادثة Kfriends Chat لنظام Android
مبني بـ HTML/CSS/JavaScript و Kotlin

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ المميزات:
  • عرض المحادثات من ملفات JSON
  • واجهة داكنة أنيقة
  • دعم اللغة العربية (RTL)
  • يعمل بدون إنترنت
  • حفظ المحادثات تلقائياً

🚀 البدء السريع:

1️⃣  فتح المشروع:
    • افتح Android Studio
    • File → Open
    • اختر مجلد KfriendsChatApp

2️⃣  بناء التطبيق:
    • Build → Build Bundle(s) / APK(s) → Build APK(s)
    • أو اضغط: Ctrl+Shift+B

3️⃣  انتظر النتيجة:
    • سيظهر إشعار عند الانتهاء
    • الملف: app/build/outputs/apk/debug/app-debug.apk
